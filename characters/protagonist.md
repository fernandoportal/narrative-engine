# Protagonist Guide

Use this file to define your central character with enough precision to drive consistent scenes.

## Core Fields

- Wound: unresolved past injury shaping current behavior
- Want: explicit goal the character pursues
- Need: internal shift required for transformation
- False belief: idea the character must outgrow

## Arc Checkpoints

- Opening behavior pattern
- First major compromise
- Midpoint reorientation
- Crisis decision
- Final value-based action

A strong protagonist arc changes strategy because inner belief changes, not just because events escalate.
