# Full Film Mini Example: Signal in the Fog

This example shows a compact end-to-end Narrative Engine workflow for a feature
film concept.

## Premise

A suspended harbor safety inspector must reopen a decommissioned lighthouse
network to guide a hospital ship through toxic fog before sunrise.

## How the Files Connect

1. `story/logline.md` defines the conflict, goal, and stakes in one sentence.
2. `story/synopsis-short.md` expands the arc from setup to resolution.
3. `characters/protagonist.md` and `characters/antagonist.md` define opposing
   motivations.
4. `structure/beat-sheet.md` maps major turning points.
5. `structure/scene-list.md` converts beats into actionable scenes.
6. `screenplay/opening.fountain` demonstrates scene-level execution.

## How to Study This Example

- Read files in the order above.
- Trace how one idea moves from premise to scene.
- Reuse the pattern for your own project structure.
