# Antigravity First-Use Prompts

Copy and paste one of these prompts in Antigravity.

## Prompt A - Interactive first story onboarding

Use this if you want AI to guide you step by step.

```text
Use `onboarding/FIRST_STORY_AI.md` as your protocol.
Guide me interactively to create my first story in less than 10 minutes.
Ask one thing at a time and wait for my answer.
Always tell me exactly which file to edit.
```

## Prompt B - Direct setup without back-and-forth

Use this if you want AI to draft a starter pack immediately.

```text
Create a first-story starter for Narrative Engine using these files:
- story/logline.md
- characters/protagonist.md
- structure/beat-sheet.md
- screenplay/selected-scenes/opening.fountain

Keep it simple and coherent. Then summarize what you changed.
```
