# First-Use Onboarding

This folder is the fastest way to start Narrative Engine for the first time in
Antigravity.

## Choose your path

- **Human guide:** `onboarding/FIRST_STORY_HUMAN.md`
- **AI guide:** `onboarding/FIRST_STORY_AI.md`
- **Copy-paste prompts:** `onboarding/ANTIGRAVITY_PROMPTS.md`

## Outcome (under 10 minutes)

You will create:

1. A one-sentence story idea (`story/logline.md`)
2. A protagonist profile (`characters/protagonist.md`)
3. A simple structure (`structure/beat-sheet.md`)
4. A first scene (`screenplay/selected-scenes/opening.fountain`)
